# Reservas Online

## Visão Geral

Projeto desenvolvido para facilitar reservas de determinados espaços e evitar conflito de horários.

## Tecnologias Utilizadas

- Node.js
- Express
- EJS
- bcrypt
- dotenv
- express-ejs-layout
- jsonwebtoken
- pg
- sequelize
